# 🌐 Relationship Tracker App

## 📖 Description
A modern and interactive web application to manage and maintain personal relationships. Users can track communication frequency, set goals, and log interactions like calls, texts, and video chats. The app provides a timeline view and analytics to help users stay connected with friends effectively.

---

## 🚀 Technologies Used
- **Next.js** (App Router)
- **React.js**
- **Tailwind CSS**
- **Context API** (State Management)
- **Recharts** (Data Visualization)
- **React Icons**

---

## ✨ Key Features

### 1. 📊 Timeline Tracking
- Log interactions (Call, Text, Video)
- Automatically adds entries with date and activity
- Filter timeline by activity type

### 2. 👥 Friend Management
- View detailed friend profiles
- Track days since last contact
- Set communication goals and reminders

### 3. 📈 Analytics Dashboard
- Visualize interaction data using charts
- See percentage of Call, Text, and Video activities
- Real-time updates based on user actions
